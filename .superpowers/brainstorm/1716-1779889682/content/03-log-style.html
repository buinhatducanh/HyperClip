Chọn style cho ActivityLog

1

Timeline (Ant Design style)

Dot indicator + đường kết nối dọc. Clean, hiện đại, phù hợp light theme.

2

Terminal Log

Dark terminal style, timestamp + status tag + message. Phân biệt rõ với UI chính.

3

Hybrid

Timeline cho activity chính + mini-terminal cho log chi tiết. Kết hợp cả hai.
