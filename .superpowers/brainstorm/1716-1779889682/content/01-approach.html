HyperClip — Redesign UI

Phân tích vấn đề hiện tại và định hướng thiết kế mới

Vấn đề hiện tại

⚠️ Inline styles

  • 100% component dùng style={{}} — zero design system
  • Không color tokens, không spacing scale
  • Theme switch cần sửa từng file

⚠️ Visual thiếu cấu trúc

  • Card bg #FFFFFF trên nền #F5F5F5 — thiếu depth
  • Border #E0E0E0 quá nhạt — khó thấy ranh giới
  • Status colors chói trên nền sáng

⚠️ Typography hỗn loạn

  • 8 cấp font-size không có logic
  • Spacing tùy tiện (padding 8, 10, 12, 14, 16)
  • Thiếu hierarchy giữa sections

Hướng tiếp cận

A

Minimal Refine

Giữ nguyên layout, chỉ chuẩn hóa color tokens + spacing + borders. Dùng Tailwind classes thay inline styles cho file chính. Ít rủi ro, nhanh (2-3 sessions).

B

Design System

Xây dựng CSS variables đầy đủ (colors, spacing, typography, shadows) + utility components (Card, Badge, Button). Chuyển dần component quan trọng nhất. Trung bình (4-6 sessions).

C

Full Redesign

Làm lại toàn bộ UI: layout mới, design system, component library, animations. Component nào xấu viết lại từ đầu. Tốn thời gian nhất (8-12 sessions) nhưng kết quả đẹp nhất.
