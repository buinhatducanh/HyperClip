Bố cục Desktop 3-panel:

  • Top bar: Download config + Auto-render toggle + System health (GPU/RAM bars)
  • Panel 1 (~160px): Channels + detection health
  • Panel 2 (flex): Video queue với tabs + search
  • Panel 3 (~220px): Settings — auto-render config (resolution, fps, parts), per-channel override, download, storage, system info
  • Bottom bar (80px): Activity log + errors badge + health status dots
