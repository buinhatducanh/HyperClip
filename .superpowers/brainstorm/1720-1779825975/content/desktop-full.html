Right Panel (260px) — 10 sections:

  1. AUTO RENDER — toggle + resolution + fps + parts + minutes/part
  2. CHANNEL OVERRIDE — override global per channel
  3. DOWNLOAD — quality, trim, concurrent, min/max duration
  4. PROXY — toggle + host/port/auth
  5. DETECTION — Innertube/OAuth health, session health %
  6. STORAGE — disk usage bar, paths, clear cache
  7. SYSTEM — GPU/CPU/RAM/workers
  8. LOGS — export button
  9. UPDATE — version + check for update
  10. ADVANCED — Sessions / OAuth / API Keys / Diagnostics → mở modal hoặc /settings
