Panel Layout cho Dashboard mới

Bỏ manual editor → 100% auto-render. Đưa settings lên trang chủ.

Bạn đang đề cập đến các panel. Tôi thấy message bị cut ở "Pannel cài...", hãy chọn các panel bạn muốn có trên dashboard:

1

Kênh đang theo dõi

Danh sách channel + auto-render toggle + config render (FPS, resolution, split parts)

2

Detection Status

Innertube/OAuth health, poller status, sessions

3

Download Config

Quality, trim limit, concurrent downloads, proxy

4

System Monitor

CPU/GPU/RAM, workers, network

5

Storage

Disk usage, paths, clear cache

6

Activity Log

Pipeline events, download/render progress

7

Auth & API Keys

Sessions, OAuth projects, keys

8

Rendered Videos

Danh sách video đã render xong

Bạn cũng có thể mô tả thêm nếu muốn panel khác hoặc cấu trúc khác.
